# Mihai's Lego Tetris

A mobile-first browser Tetris game for Mihai.

## Local Run

```bash
python3 -m http.server 3010
```

Then open:

```text
http://localhost:3010
```

## Controls

- Tap board: rotate
- Swipe left/right: move
- Swipe down/flick down: drop
- Buttons: left, rotate, right, drop
- Mute button in the top HUD
